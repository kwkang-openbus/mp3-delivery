# PRD: 외국어 교재 MP3 자동 배달 서비스

> **문서 버전**: 1.0
> **작성일**: 2026-04-02
> **프로젝트**: Bus-Ton 해커톤 (AI Vibe Coding)

---

## 1. 프로젝트 개요

### 1.1 배경
외국어 학습 시 교재 부교재(MP3 음원)를 구하기 어렵거나, 다운로드 과정이 복잡한 경우가 많다. 이메일 한 번으로 원하는 교재의 MP3를 받을 수 있는 서비스를 제공한다.

### 1.2 목표
- 사용자가 교재를 검색하고, 이메일 입력만으로 MP3 파일을 받을 수 있는 웹 서비스
- 관리자가 교재와 MP3를 등록/관리할 수 있는 대시보드

### 1.3 대상 사용자

| 사용자 | 설명 |
|--------|------|
| **학습자** | 외국어(일본어/중국어) 교재 MP3가 필요한 학생·직장인 |
| **관리자** | 교재 데이터와 MP3 파일을 등록·관리하는 운영자 |

---

## 2. 기능 요구사항

### 2.1 사용자 기능

#### FR-U01: 교재 검색
- 키워드로 교재 검색 (제목, 출판사, 설명)
- 200ms 디바운스 적용 실시간 검색

#### FR-U02: 언어 필터
- 전체 / 일본어 / 중국어 필터 칩
- 검색과 필터 동시 적용 가능

#### FR-U03: 교재 선택
- 카드 클릭 시 선택 상태 표시 (하이라이트)
- 하단 이메일 입력 패널 슬라이드업

#### FR-U04: MP3 이메일 발송
- 이메일 입력 후 "전송하기" 클릭
- 이메일 형식 검증 (프론트엔드 + 백엔드)
- 성공/실패 토스트 알림

#### FR-U05: 반응형 UI
- PC(1200px+) / 모바일(480px-) 대응
- 3단계 진행 표시 (검색 → 선택 → 전송)

---

### 2.2 관리자 기능

#### FR-A01: 로그인/로그아웃
- 아이디/비밀번호 세션 인증
- 세션 만료: 1시간

#### FR-A02: 대시보드
- 총 교재 수, 오늘 발송 수, 누적 발송 수
- 최근 발송 이력 10건

#### FR-A03: 교재 등록
- 교재명(필수), 언어(필수), 레벨, 출판사, 설명
- MP3 파일 업로드 (최대 100MB, audio/mpeg)

#### FR-A04: 교재 수정
- 모든 메타데이터 수정 가능
- MP3 파일 교체 시 기존 파일 자동 삭제

#### FR-A05: 교재 삭제
- 교재 삭제 시 MP3 파일도 함께 삭제

#### FR-A06: 발송 이력 조회
- 전체 이력 테이블 (최근 200건)
- 교재명, 이메일, 시간, 상태(성공/실패), 오류 메시지

## 3. 데이터 모델

### 3.1 교재 (Textbook)

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| id | number | Y | 자동 증가 PK |
| title | string | Y | 교재명 |
| language | string | Y | `japanese` 또는 `chinese` |
| level | string | N | 레벨 (N5, HSK 1급 등) |
| publisher | string | N | 출판사 |
| description | string | N | 교재 설명 |
| mp3_filename | string | N | 저장된 MP3 파일명 |
| mp3_size | number | N | 파일 크기 (bytes) |
| created_at | string | Y | 생성일시 |
| updated_at | string | Y | 수정일시 |

### 3.2 발송 이력 (History)

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| id | number | Y | 자동 증가 PK |
| textbook_id | number | Y | 교재 ID |
| textbook_title | string | Y | 교재명 (비정규화) |
| email | string | Y | 수신 이메일 |
| status | string | Y | `success` 또는 `failed` |
| error_msg | string | N | 실패 시 오류 메시지 |
| sent_at | string | Y | 발송 일시 |

## 4. API 명세

### 4.1 공개 API

| Method | Endpoint | 설명 |
|--------|----------|------|
| GET | `/api/textbooks?q=&language=` | 교재 검색/필터 |
| POST | `/api/send` | MP3 이메일 발송 |

### 4.2 관리자 API

| Method | Endpoint | 설명 |
|--------|----------|------|
| POST | `/api/admin/login` | 로그인 |
| POST | `/api/admin/logout` | 로그아웃 |
| GET | `/api/admin/me` | 인증 상태 확인 |
| GET | `/api/admin/stats` | 대시보드 통계 |
| GET | `/api/admin/textbooks` | 교재 전체 목록 |
| POST | `/api/admin/textbooks` | 교재 등록 (multipart) |
| PUT | `/api/admin/textbooks/:id` | 교재 수정 |
| DELETE | `/api/admin/textbooks/:id` | 교재 삭제 |
| GET | `/api/admin/history` | 발송 이력 |

## 5. 비기능 요구사항

### 5.1 보안
- IP 기반 Rate Limiting: 15분당 5회 (`/api/send`)
- 관리자 세션 인증 (express-session)
- 이메일 형식 검증 (정규식)
- MP3 파일 타입 검증 (MIME type)
- 환경변수로 민감 정보 분리 (.env)

### 5.2 성능
- 검색 디바운스 200ms
- JSON 파일 기반 DB (소규모 운영 적합)

### 5.3 운영 환경
- Node.js 18+
- 포트: 3001 (기본값)
- 외부 의존: Resend API (이메일 발송)

## 6. 제약사항

| 항목 | 내용 |
|------|------|
| 이메일 발송 | Resend 무료 플랜: 계정 소유자 이메일만 수신 가능 (도메인 인증 시 해제) |
| 파일 크기 | MP3 업로드 최대 100MB |
| 동시성 | 단일 프로세스, JSON 파일 DB (고부하 미지원) |
| 언어 | 일본어/중국어만 지원 (확장 가능) |
